# 추가 기능

- 숫자 맞추기 게임에서 취소를 누를 때와 범위 외의 숫자, 숫자 외의 문자에 대한 예외 케이스 처리 (정규식 사용)
- 성적 계산기에서 취소를 누를 때와 범위 외의 숫자, 숫자 외의 문자에 대한 예외 케이스 처리 (정규식 사용)
- 실시간 날씨 정보의 온도와 습도에 따른 아이콘 변화 (삼항 연산자 사용)
- 회원 가입 비밀번호 제약 강화 (정규식 사용)
- 회원 가입 완료 페이지에 홈 버튼 추가
- `index.html`에 실시간 위치 정보 추가를 위한 `location.js` 생성 (kakao JS API key 사용)
  - 교육용 맥북으로 GPS 기능이 안되는 경우 임의 좌표를 넣어 사용하도록 설계 (가능한 기기에서의 정상 작동 확인)
  - kakao JS API key의 경우 등록된 페이지 외에 사용이 불가능해 현재는 로컬에서만 사용할 수 있어 공개, 평가 후에 내릴 예정
- `index.html`에 이달의 베스트 컷의 이미지를 랜덤하게 보여주도록 `imageRandom.js` 생성
- 내 가방 보기 기능을 개선해 정해진 물건 리스트에서 물건들을 챙기거나 꺼재고 현재 챙긴 물건들을 볼 수 있도록 `bag.js` 수정
